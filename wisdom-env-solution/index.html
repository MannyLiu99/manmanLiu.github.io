智慧环保
覆盖生态环境全应用，为中国生态保驾护航
环保产业发展事关国计民生，是我国重要的战略性产业。 虹川科技智慧环保解决方案旨在打造集环境感知、分析、管控、治理于一体的全产业链创新综合体， 整合联合环保产业链的创新资源，综合运用环境物联网、人工智能、高密度、全覆盖监测和大数据分析等技术手段， 以源解析加减排规划制定为科技指引，结合大数据应用分析，精准找源，科学治理，实现区域环境持续达标治理与保持经济发展平衡。

产品应用

PRODUCT APPLICATION
  • 大气环境监测企业电量监控AI图像识别污染源特征分析空气质量预报
  • 环境综合分析网格监管重污染天气应对空气质量评估节能减排评估

顶层架构

TOP ARCHITECTURE
1个数据中心3大保障体系5大技术支撑N个应用服务

方案特点

PROGRAM FEATURES
测
环境感知物联网
从顶层规划的理念出发，构建符合城市实际和特色的一整套智慧城市的综合信息服务体系。
融
数据共享融合
将大气环境大数据的建设及应用融入以人为核心的产业发展、城市治理及大气环境改善中，为各环保相关方提供环保大数据服务。
管
智能闭环协同管理
实现基于大数据的监管预警、研判溯源、决策指挥、治理处置、评估考核等全过程可视化、痕迹化、智能化的闭环管理。
治
多元共治，共治多源
搭建公众等多元主体共同参与到环境治理中，推动构建多主体联动、多对象共治、多方式优化的多元共治环保格局。
