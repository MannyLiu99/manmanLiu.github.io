智慧水务
供水排水水务全流程管理，为城市生活供给生命源泉
水是生命之源，水务是国民经济的命脉。虹川科技智慧水务方案全面应用数字化综合信息新技术手段， 创新性的“一张图”、“一个中心数据库”的建设方案， 将物联网和大数据技术应用于供排水循环全过程监控，建立覆盖“点---线---面”的水环境监管体系， 实现“信息全面感知、数据综合分析、高度智慧决策”的智慧化应用模式，保障国民用水安全。

  • 防汛抗旱SCADA运行调度河湖长制管理海绵城市污水处理监测

方案特点

PROGRAM FEATURES
开源节流
从水源地到水龙头，从水龙头到自然环境，整个流程保障水质的安全性，实时掌控水质变化情况，及时发现水质问题，问题发生点，并进行问题追踪，充分保障居民用水安全。
产销智控
从供水角度，达到生产工艺过程自动且可控。从用水角度，做到实时掌控用水情况，并可对用水情况进行预测，从而做到供水和用水的平衡化和智能化。
数治全程
充分调用各类水务相关数据，实现从水源地—输水管网—水厂—供水站—用户—排水管—污水处理厂—排水口全过程数据收集融合应用。
